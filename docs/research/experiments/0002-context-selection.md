# Experiment 0002: Context Selection

## Question

Can Token Shuffle select engineering context based on relevance, authority, velocity, and current task?

## Success Criteria

- Context selection includes relevant high-authority artifacts.
- Volatile observations do not overwhelm stable decisions.
- Model prompts become more consistent across providers.

## Failure Criteria

- Context selection becomes unpredictable.
- Irrelevant documents increase token usage without improving quality.
