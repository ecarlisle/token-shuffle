# Engineering Reviews

Engineering Reviews are immutable assessments of the repository performed after a milestone or release.

They evaluate:

- product clarity
- documentation quality
- architecture
- implementation consistency
- developer experience
- contributor readiness
- open-source readiness

Unlike ADRs, which explain why architectural decisions were made, Engineering Reviews evaluate the repository after implementation and produce Engineering Review Packets for future work.

## Lifecycle

```text
Vision
  -> ADRs
  -> Implementation
  -> Release Validation
  -> Engineering Review
  -> Engineering Review Packets
  -> Implementation
  -> Next Release
```
